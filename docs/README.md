# ユーティリティツール集 - 設計ドキュメント

## プロジェクト概要

軽量でミニマルなデザインのReactベースユーティリティツールサイト。
タイマー、テキストツールなどの日常的に使える便利なツールを提供します。

## 設計方針

- **軽量性**: バンドルサイズ < 100KB (gzip圧縮後)
- **シンプルさ**: 必要最小限の機能とUI
- **高速**: 初回読み込み < 1秒
- **使いやすさ**: 直感的なインターフェース
- **オフライン対応**: クライアントサイドのみで動作

## 技術スタック

| 分類 | 技術 | 理由 |
|------|------|------|
| フレームワーク | Vite + React 18 | 高速ビルド、最小構成 |
| 言語 | TypeScript | 型安全性、開発体験向上 |
| ルーティング | React Router v6 | SPA標準ライブラリ |
| スタイリング | CSS Modules | ゼロランタイム、軽量 |
| ビルド | Vite | 最速のHMR、最適化済み |
| デプロイ | Vercel | 自動デプロイ、高速配信 |

## ツール一覧

### タイマー系
- カウントダウンタイマー - 指定時間のカウントダウン
- ストップウォッチ - 時間計測とラップタイム
- ポモドーロタイマー - 25分作業 + 5分休憩サイクル

### テキストツール
- 文字数カウンター - 文字数、単語数、行数カウント
- テキスト diff - 2つのテキストの差分表示
- ランダム文字列生成 - パスワード等の生成

### その他ツール
- QRコード生成 - URL等をQRコードに変換
- パスワード生成 - セキュアなパスワード生成
- カラーピッカー - HEX/RGB/HSL変換

## パフォーマンス目標

| 指標 | 目標値 |
|------|--------|
| 初回読み込み時間 | < 1秒 |
| バンドルサイズ (gzip) | < 100KB |
| Lighthouse Performance | 95+ |
| Time to Interactive | < 500ms |
| First Contentful Paint | < 600ms |
