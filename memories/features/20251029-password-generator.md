# パスワード生成

## 日付
2025-10-29

## 実装目的
セキュアなパスワードを生成する。ランダム文字列生成機能をベースに、パスワード特有の要件（強度表示、除外文字など）を追加した専用ツール。

## 画面構成・UI
- パスワード長さスライダー（8〜64文字）
  - デフォルト: 16文字
- 文字種チェックボックス
  - 大文字（A-Z）
  - 小文字（a-z）
  - 数字（0-9）
  - 記号（!@#$%^&*など）
- 除外する文字の入力（オプション）
  - 例: 紛らわしい文字（0, O, l, I, 1）を除外
- パスワード強度表示
  - 弱い（赤）
  - 普通（黄）
  - 強い（緑）
- 生成されたパスワード表示
- コピーボタン
- 再生成ボタン

## データ構造

### パスワード強度判定
```typescript
type PasswordStrength = 'weak' | 'medium' | 'strong'

interface PasswordStrengthResult {
  strength: PasswordStrength
  score: number // 0-100
}
```

## 技術的な設計判断

### ランダム文字列生成との違い
- パスワード生成はランダム文字列生成の特殊版
- 既存のrandomStringUtilsを再利用
- パスワード強度計算を追加
- UIはパスワード生成に特化した表現

### パスワード強度の計算
簡易的な強度判定:
1. 長さのスコア（8文字未満=弱、8-15文字=中、16文字以上=強）
2. 文字種の多様性（1種類=弱、2種類=中、3種類以上=強）
3. 総合スコアで判定

### 除外文字機能
- ユーザーが指定した文字をcharsetから除外
- 紛らわしい文字（0OlI1）を簡単に除外できるプリセットボタン

## リスク・注意点
- ランダム文字列生成と機能が重複する
  - → パスワード生成は強度表示など付加価値があるため別ツールとして提供
- 除外文字を多く指定しすぎるとエントロピーが下がる
  - → UIで警告は出さないが、強度表示で反映される
