# 文字数カウンター

## 日付
2025-10-28

## 実装目的
テキストの文字数、単語数、行数をリアルタイムでカウントする機能を提供する。
ライター、翻訳者、学生などがテキストの分量を確認するために使用。

## 画面構成・UI
- テキスト入力エリア（textarea）
- カウント結果表示エリア
  - 文字数（全体）
  - 文字数（スペース除く）
  - 単語数
  - 行数
- クリアボタン
- ミニマルなデザイン、視認性の高いカウンター表示

## データ構造

### CharCounter コンポーネント
```typescript
interface TextStats {
  chars: number;          // 全文字数
  charsNoSpaces: number;  // スペース除く文字数
  words: number;          // 単語数
  lines: number;          // 行数
}
```

## 実装ファイル
- src/pages/TextTools/CharCounter.tsx - 文字数カウンターUI
- src/pages/TextTools/CharCounter.module.css - スタイル
- src/utils/textUtils.ts - テキスト処理ユーティリティ

## 実装方針
1. useMemoで計算結果をキャッシュし、パフォーマンス最適化
2. 単語数は英語・日本語両対応
   - 英語: スペース区切り
   - 日本語: 文字単位でカウント（または形態素解析なしで簡易的に）
3. リアルタイムカウント（onChange）
4. LocalStorageへの保存は不要（シンプルさ優先）

## リスク・注意点
- 大量テキスト: 10,000文字以上でもパフォーマンス維持
- 単語数の定義: 英語と日本語で異なるため、シンプルな実装に
- バンドルサイズ: 外部ライブラリは使わず、標準APIのみ使用
